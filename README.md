# 1D_Heat-Equation
Numerical Simulation of 1D Heat Equation using Finite Difference Method

This project solves the one-dimensional heat equation using the Finite Difference Method (FDM). The spatial and temporal domains are discretized, and an explicit scheme is implemented to simulate heat diffusion in a rod over time under given initial and boundary conditions.

The project emphasizes numerical stability by applying the CFL condition to select appropriate time and space step sizes. The implementation is done in Python using NumPy for computations and Matplotlib for visualization.

Temperature evolution is visualized through plots, providing insights into heat propagation behavior.
